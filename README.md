# Doopmarketeer

Finds Dooplicator and Doopmarket history of a given Ethereum Address. Using doopmarketeer-api repo endpoints to pull data. Made with React/Chakra.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```
